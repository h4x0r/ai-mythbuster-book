# AI Mythbusters Book Cover Specifications

## Image Requirements
- **Dimensions**: 2100 x 3000 pixels
- **Filename**: `title_page.png` or `title_page.jpg`
- **Location**: Place in this folder (`manuscript/images/`)

## Design Recommendations

### Title
**AI Mythbusters**
- Large, bold, modern sans-serif font
- High contrast for readability

### Subtitle (Optional)
**What AI Actually Is, What It Can Really Do, and How to Use It Effectively**

### Visual Theme Suggestions

**Option 1: Tech Minimalist**
- Clean gradient background (deep blue to purple)
- Geometric circuit-like patterns
- Modern, professional aesthetic

**Option 2: Myth-Busting Visual**
- Shattered glass or breaking chains imagery
- Contrast between dark myths and light truth
- Bold, impactful typography

**Option 3: Practical Focus**
- Split design: hype/myths on one side, reality on the other
- Tool icons or code snippets as subtle background
- Professional yet approachable

### Color Palette Ideas
- **Primary**: Deep navy blue (#1a1f3a) or charcoal (#2c3e50)
- **Accent**: Electric blue (#00a8ff), cyan (#00d2d3), or amber (#f39c12)
- **Text**: White (#ffffff) or light gray (#ecf0f1)

### Typography
- **Title font**: Bold sans-serif (Montserrat Bold, Bebas Neue, Roboto Bold)
- **Subtitle font**: Clean sans-serif (Open Sans, Lato, Inter)

## Quick Creation Tools

### Free/Easy Options
1. **Canva** (canva.com) - Has book cover templates
2. **Adobe Express** - Free tier available
3. **Placeit by Envato** - Book cover maker

### Professional Options
1. **Fiverr** - Hire a designer ($20-100)
2. **99designs** - Professional design contest
3. **Adobe Photoshop/Illustrator** - Full control

### AI Image Generation
1. **Midjourney** - High quality, subscription required
2. **DALL-E 3** - Via ChatGPT Plus or API
3. **Stable Diffusion** - Free, self-hosted option

## Example Prompt for AI Image Generation

"Create a professional book cover for 'AI Mythbusters', dimensions 2100x3000 pixels, modern tech aesthetic with deep blue gradient background, bold white typography, subtle circuit board patterns, minimalist design, high contrast, suitable for a practical guide about artificial intelligence reality vs hype"

## After Creating Your Cover

1. Save as `title_page.png` (recommended) or `title_page.jpg`
2. Place in this `manuscript/images/` folder
3. Verify dimensions are exactly 2100 x 3000 pixels
4. Check file size (aim for under 5MB)
